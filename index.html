Первый уровень

D3 реализует подход, называемый fluent interface. При чтении кода он выглядит как цепочка методов. Каждый метод вызывается на объекте, который вернул предыдущий метод. Чтобы код было удобно читать, каждый вызов располагается на отдельной строчке.
Например, создадим линию синего цвета:
d3.select('body') // выбор в документе body
 .append('svg') // добавление в body svg-контейнера, система координат начинается с верхнего левого угла и идет вправо вниз
 .append('line') // добавление фигуры линия
 .attr('x1', 20) // задание x-координаты первой точки
 .attr('y1', 30) // задание y-координаты первой точки
 .attr('x2', 300) // задание x-координаты второй точки
 .attr('y2', 300) // задание y-координаты второй точки
 .style('stroke', 'blue') // задание цвета линии
 .style('stroke-width', '2') // задание толщины линии
